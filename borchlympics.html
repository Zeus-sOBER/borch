2-minute free setup:
1. Go to console.firebase.google.com
2. Add project → name it anything → Continue
3. Left sidebar → Build → Realtime Database
4. Create Database → any location → Start in test mode
5. Copy the URL at the top (e.g. https://project-rtdb.firebaseio.com)
6. Paste it above and hit Connect!
Your URL is saved locally — you won't need to re-enter it next time.
